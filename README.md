# Portfolio

Simple GitHub Pages portfolio for Puja Raghav.

## Structure

- `index.html`: landing page
- `styles.css`: page styling
- `script.js`: small reveal animations and footer year
- `resources/`: supporting assets and reference material
- `docs/`: portfolio docs and research notes
- `certificates/`: certifications and proofs
- `capabilities/`: capability statements and skill summaries

## Deployment

This repository is designed to work directly with GitHub Pages from the root branch.

## Notes

The first version uses publicly accessible, verifiable profile details. If you want to refine the copy further, update `docs/profile-analysis.md` and then adjust the landing page text.
